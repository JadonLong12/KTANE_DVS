# Keep Talking And Nobody Explodes

A bomb defusal game inspired by the [video game](https://en.wikipedia.org/wiki/Keep_Talking_and_Nobody_Explodes) of the same name.

You have 5 minutes to successfully defuse the bomb. one player with the instructions, will instruct the other player in front of the bomb on how to succesfully disarm each module.

On the right side of the timer are three red LEDs. These are strikes. Each time you make a mistake, one LED will light up. After three strikes, the bomb explodes.

Once all modules are disarmed, the bomb is defused!
